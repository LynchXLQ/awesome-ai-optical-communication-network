## How to Contribute

### Adding a Paper

1. Fork this repository
2. Add the paper to the appropriate section in `survey.md`
3. If it's a survey paper, also add it to the README
4. Submit a Pull Request

### Paper Entry Format

```markdown
- Author, et al. **Paper Title**. Venue Year. [Link](URL)
  - Brief description (optional)
```

### Requirements

- Papers should be peer-reviewed (journal/conference) or notable arXiv preprints
- Include a working link (DOI preferred)
- Categorize correctly by topic and year

### Adding Tools/Datasets

Follow the same process. Include:
- Tool/dataset name
- Brief description
- Link to official source

## Code of Conduct

- Be respectful and constructive
- Focus on quality over quantity
- Verify links before submitting

## Questions?

Open an issue if you have questions or suggestions!
